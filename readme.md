# Custom DAP Project Notes

Welcome to my Digital Audio Player (DAP) project repository.  
This repo contains organized notes, definitions, diagrams, and reference PDFs for both hardware and software components used in the build.

---
## Repository Structure

```
DAP/  
├─ 00_INDEX.md # Project overview, current status, active decisions  
├─ 01_REFERENCES/ # Notes, datasheets, PDFs  
├─ 02_HARDWARE/ # Hardware design notes, schematics, measurements  
├─ 03_SOFTWARE/ # Firmware, OS notes, configuration, scripts 
├─ 04_GLOSSARY/ # definitions,
```


---

## Contents Overview

### 00_INDEX.md
- High-level project overview
- Current focus and blockers
- Key constraints and design decisions

### REFERENCES/
- **Glossary / Definitions** – Key terminology for DAPs, DACs, SoCs, I²S, amps, etc.
- **Datasheets & PDFs** – Reference material for ICs, DACs, SoCs, and other components
- **Hardware & Software Notes** – Notes, diagrams, and measurements supporting the build

### HARDWARE/
- Design notes for DACs, amps, CPU/SoC selection, power management
- Block diagrams, schematics, PCB revisions
- Measurements, test results, and design tradeoffs

### SOFTWARE/
- Firmware design and architecture
- OS setup and configuration notes
- Scripts and helper tools for testing, flashing, and measurement

## How to Use This Repo
- Browse the `REFERENCES/` folder for definitions, datasheets, and PDFs.
- Follow the notes in `HARDWARE/` and `SOFTWARE/` to understand design decisions.
- Check `00_INDEX.md` first for project status and current focus.

## Purpose
This repository is designed as a **living documentation** of the DAP build process:
- Track decisions and reasoning
- Maintain organized notes and references
- Preserve measurements, schematics, and configurations
- Provide a public record of the design process for others to learn from
