The **Raspberry Pi 5** is the latest generation of Raspberry Pi boards. It features a **64-bit quad core ARM Cortex-A76** processor running at $2.4 GHz$. It delivers a 2-3x increase in CPU performance relative to the Pi 4 board. 

---
## Specifications
- **Processor:**  Broadcom BCM2712 2.4GHz quad-core 64-bit Arm Cortex-A76 CPU, with Cryptographic Extension, 512KB per-core L2 caches, and a 2MB shared L3 cache
- **Features:**
	- LPDDR4X-4267 SDRAM (multiple configs for 1GB, 2GB, 4GB, 8GB, and 16GB)
	- Dual-band 802.11ac Wi-Fi
	- Bluetooth 5.0 / Bluetooth Low energy
	- 2 X USB 3.0 ports, supporting 5Gbps operations
	- 2 X USB 2.0 ports
	- Gigabit Ethernet, with PoE+ support (requires separate PoE+ HAT tho)
	- 40 pin GPIO header
	- microSD card slot
	- 5V/5A DC power via USB-C, with PD support
	- Power button
